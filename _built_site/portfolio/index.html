// Things I've Made...

Oxylus

  • An in-progress video game developed in Unreal Engine.
  • A third-person exploration game in a stylized open world.
  • Dense foliage with custom shaders and optimized models.
  • Procedurally animated creatures that roam the world.

Wikar // An Augmented Reality App

  • An augmented reality application developed using Unity for iOS and Android phones.
  • Dynamically loads content from a web server and displays it in augmented reality.
  • Capable of utilizing 3D models, images, and audio files.
  • Developed in collaboration with Colter Wehmeier at the National Center for Supercomputing Applications in the Advanced Visualization Lab.

Soul Chiseler // A video game made in UE4

  • A video game developed in Unreal Engine over five days for the Unreal Summer Jam.
  • A top-down fighting game with a dash mechanic and multiple enemy types.
  • Follows a sculptor on their journey to immortalize their artistic vision
  • All art, shaders, animations and code made by me.

Experiments

Harpoon Highway

  • A video game developed in Unreal Engine over two days for the GMTK 2020 Game Jam.
  • A stylish top-down racing game where the only way to steer your car is with a giant harpoon.

Physarum Simulation

  • Physarum slime mold simulation create in Unreal Engine using Niagara.

Stylized Desert

  • A cel shaded desert environment developed in Godot

Hollow Knight Card

  • A virtual trading card created in Blender.

Raymarching Fractals

  • A shader that renders the Mandelbulb Fractal in Unreal Engine
  • Uses Phong shading to render the fractal as a surface.

Vine Generator

  • A system for quickly adding vines to 3D environments in Unreal Engine.
  • The vines conform to the surfaces of 3D objects and spread out randomly.

Tankuloid

  • An online multiplayer game developed using Godot Engine.
  • Battle in tanks with up to 8 friends.
