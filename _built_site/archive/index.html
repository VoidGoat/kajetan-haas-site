Wikar: Augmented Reality

  • An augmented reality application developed using Unity for iOS and Android phones.
  • Dynamically loads content from a web server and displays it in augmented reality.
  • Designed to allow researchers to quickly visualize their data in augmented reality.
  • Capable of utilizing 3D models, images, and audio files.
  • Developed in collaboration with Colter Wehmeier at the National Center for
    Supercomputing Applications in the Advanced Visualization Lab.

Unity, C#, iOS, Android
